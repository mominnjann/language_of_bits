Contributing to Language_of_Bits

Thanks for your interest in improving this project! Small, clear contributions are welcome.

Guidelines
- Keep explanations short, accurate, and beginner-friendly.
- Use examples and small code/bit patterns where helpful.
- Add images to `assets/images/` and reference them from the docs.

Workflow
1. Fork the repository and create a feature branch.
2. Make your changes and add/update tests or examples if relevant.
3. Open a pull request with a short description of what you changed and why.

Style
- Prefer simple language and consistent capitalization for headings.
- Use Markdown for docs; include code blocks for binary examples.

License
All contributions are licensed under the project license (see `LICENSE`).

Contact
Open issues for discussion before large changes.
